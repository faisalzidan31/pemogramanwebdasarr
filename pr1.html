Cerita Saya

Nama saya adalah Faisal zidan. Saya berasal dari daerah kabupaten Magelang tepatnya didesa Tlogorejo kecamatan Grabag. Saya lahir pada tahun 2006 tepatnya tanggal 31 Agustus 2006, saya sekarang masih berusia 18 tahun, tahun ini berusia 19 tahun. saya sekarang statusnya sebagai mahasiswa tewknik informatika di kampus STMIK Bina Patria

Saya berkuliah di STMIK Bina Patria masih semester 2 belum ada setahaun. Saya sebelumnya berasal dari SMA N 1 Grabag, dengan jurusan ips. sebelum ke SMA N 1 Grabag saya bersekolah di SMP N 1 Grabag. Sebelum ke SMP N 1 Grabag saya sekolah di SD N Tlogorejo. Saya ssat itu langsung masuk SD tidak TK dan PAUD.

Saya mengambil jurusan teknik informatika karena saya suka dengan teknologi kumputer. Serta prospek kerja yang sangat luas. Saya dikampus STMIK Bina Patria mengikuti himpunan dan UKM ,yaitu himpunan mahasiswa teknik informatika, UKM Esport dan UKM multi media. saya diHIMTI masuk dalam divisi riset dan pengembangan teknologi, untuk UKM Esport saya sebagai pemain dan untuk UKM multimedia saya masuk kedivisi vidiografi
